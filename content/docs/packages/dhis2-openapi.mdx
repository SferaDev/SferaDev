A type-safe API client for the [DHIS2](https://dhis2.org/) health information system API, auto-generated from the official OpenAPI specification with full TypeScript support.

## Features

- **Type-Safe** - Full TypeScript support with auto-generated types from OpenAPI spec
- **Multi-Version** - Support for DHIS2 API v40 and v41
- **Auto-Updated** - Regular regeneration from upstream OpenAPI specs

## Installation

<CodeGroup>


</CodeGroup>

## Usage

### Basic Usage

```typescript
import { createClient } from "dhis2-openapi";

const client = createClient({
  baseUrl: "https://play.dhis2.org/40/api",
  headers: {
    Authorization: `Basic ${Buffer.from("admin:district").toString("base64")}`,
  },
});

// Get organisation units
const orgUnits = await client.getOrganisationUnits();

// Get data elements
const dataElements = await client.getDataElements();

// Get indicators
const indicators = await client.getIndicators();
```

### Type Exports

```typescript
import type { Types } from "dhis2-openapi";

// Use TypeScript types
type OrganisationUnit = Types.OrganisationUnit;
type DataElement = Types.DataElement;
type Program = Types.Program;
```

## API Reference

The client exposes DHIS2 API endpoints including:

- **Organisation Units** - Organizational hierarchy management
- **Data Elements** - Data element definitions
- **Indicators** - Health indicator definitions
- **Programs** - Program (tracker) configuration
- **Data Sets** - Aggregate data collection
- **Analytics** - Data analysis and reporting
- **Users** - User management
- **And more...**

For the complete API reference, see the [DHIS2 Web API documentation](https://docs.dhis2.org/en/develop/using-the-api/dhis-core-version-master/introduction.html).
